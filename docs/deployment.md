# CampusConnect Deployment Guide

## Frontend

* React + Vite
* Deployment Platform: Vercel

## Backend

* Node.js + Express
* Deployment Platform: Railway

## Database

* MongoDB Atlas

## Environment Variables

Backend:

* PORT
* MONGO_URI
* JWT_SECRET

Frontend:

* VITE_API_URL

## Docker

Run all services:

docker compose up

Stop services:

docker compose down
